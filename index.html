30 Projects with Javascript by Isioma Tàlàbí

Welcome to my '30 projects with JS' project where I aim to practice what I have learnt in Javascript over the course of my learning journey as a newbie. Here, I aim to create 30 mini-projects built with Javascript which I hope will be useful applications for anyone. So... here we go!

Basket Ball Scoreboard

This simple project employs the use of the following JS concepts:

  1. Declaring variables
  2. Adding event listeners
  3. Declaring functions
  4. Working with increments
  5. And my favourite part- Manipulating the DOM!

A Game of BlackJack

This project helped me hone my skills in the following areas:

  1. if/else conditionals
  2. Grabbing elements from the DOM with query selectors
  3. Creating a reusable functions
  4. Arrays
  5. And finally, the wonderful magical powers of
    Math.random() & Math.floor()

Password Generator

The project enabled me to practice the following skills:

  1. Getting a grip of for loops and when to use them.
  2. Working with arrays
  3. Using clipboard API in Javascript.
  4. Declaring storage variables within functions
  5. Using Styles in Javascript
  6. Understanding the concept of Promise
  7. And again, my new best friends, Math.random() & Math.floor()

Kasuwa: A Grocery List App

Definitely had fun making Kasuwa! I practiced the following skills:

  1. Something I had been dreading for a while- creating elements with Javascript.
  2. Did a thing with that edit button using event listeners and conditionals.
